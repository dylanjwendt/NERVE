A 2D Multiplayer First Game Engine

What is Nerve?

NERVE is a real-time multiplayer web game framework perfect for making io games and other multiplayer web games. Because it is built on the idea of multiplayer first this allows developers to quickly build io games without the hassle of building a game engine with functional multiplayer from the ground up. NERVE's paradigm of actors and interactions makes adding new features and game logic simple. NERVE handles all server logic and netcode including synchronization and client-side prediction, allowing developers to focus only on creating their game.
