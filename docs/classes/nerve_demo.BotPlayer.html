Class BotPlayer

this is a simple prototype version of a bot for our demo engine. it extends the Player class (which is an actor) and then passes random fake data to the engine's input handler.

Bot extends Player because in a .io game bots are supposed to appear just like players and so they have the same rules/behavior.
